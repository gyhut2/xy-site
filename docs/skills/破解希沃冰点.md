# 使用PE系统破解希沃冰点还原
## 准备
一个PE U盘，-~~你的手~~ 即可（适用于希沃4，5代机 -~~理论上应该都可以（学校只有这两种）~~），建议看完了再实践
## 原理
在PE系统中，对已冻结的磁盘进行任何操作，重启后进系统都会保存，因此得出下列方案：
## Step 1：进PE
==如果你有键盘-~~（去办公室“借”一个）~~：==
直接开机时按F7进PE即可，然后往下看Step 2

==如果你没有键盘：==
插入PE盘，转至\>设置\>更新和安全\>恢复\>高级启动\>立即重新启动
> 详见[在不使用键盘的情况下启动PE - 智教联盟论坛](https://forum.smart-teach.cn/d/999-zai-bu-shi-yong-jian-pan-de-qing-kuang-xia-qi-dong-pe)

点击 **从其他介质启动（USB，DVD）**\>选择你的PE盘，然后往下看Step 2

==如果没有这个选项- ~~（第一次肯定没有）~~==：

打开Windows，以管理员身份运行cmd，run：
```title="cmd"
reagentc /enable
```
再重做Step 1即可
## Step 2：删文件
进入PE，打开此电脑，进入系统盘
!!!warning
    有些PE上显示的C盘不是系统盘，请仔细甄别

??? info "如果不显示C盘："
    点击 开始\>更多程序\>PE工具\>显示所有磁盘分区 即可
删除以下文件：
### C:\ProgramData
删除该目录下的含有**SeewoFreeze**字段的所有文件夹
!!!warning
    Seewo文件夹不要删，里面有用户数据
### C:\users\seewo\AppData
3个子文件夹下的含**SeewoService**的文件夹删除
!!!warning
    Seewo文件夹不要删，里面有用户数据
### C:\Program Files(x86)\Seewo
名字叫SeewoService的文件夹删除
!!!warning
    **==不是删C:\Program Files(x86)\Seewo，而是删Seewo下的SeewoService文件夹==**
## Step 3：完成
重启进系统

==Congratulations！You have finished it！==
